# YARenderer

Yet another renderer built with C++ and Directx12 from scratch. 

## Features

- Forward Rendering
- Deferred Rendering (in progress)
- Image-Based Lighting
- Directional Light
  - Cascaded Shadow
  - PCSS
- Screen Space Ambient Occlusion
- Voxel-based Global Illumination (in progress)
- Post Processing
  - Motion Blur
  - Tone Mapping
  - Anti-Aliasing
    - FXAA
    - TAA
